`azure-oss/storage` is the core Azure Blob Storage SDK for PHP.

## Requirements

- PHP 8.1+
- PHP extensions:
  - `curl`
  - `json`
  - `simplexml`

## Install With Composer

```bash
composer require azure-oss/storage
```

## Next Step

Continue to [Quickstart](./quickstart) to create a service client and perform basic container/blob operations.
