`azure-oss/storage-queue` is the Azure Storage Queue SDK for PHP.

## Requirements

- PHP 8.1+
- PHP extensions:
  - `curl`
  - `json`
  - `simplexml`

## Install With Composer

```bash
composer require azure-oss/storage-queue
```

## Next Step

Continue to [Quickstart](./quickstart) to create a service client and send/receive messages.
