`azure-oss/storage-blob-flysystem` is a [Flysystem v3](https://flysystem.thephpleague.com/) adapter for Azure Blob Storage.

## Requirements

- PHP 8.1+
- `league/flysystem` `^3.0`

## Install With Composer

```bash
composer require azure-oss/storage-blob-flysystem
```

## Next Step

Continue to [Quickstart](./quickstart) to create a `Filesystem` instance and run common file operations against Azure Blob Storage.
